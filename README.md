## BBC Articles Category Classificatin

Natural Language Processing (NLP) project that classifies BBC news articles into predefined categories using TF-IDF feature extraction and machine learning.

### Dependencies

- numpy
- pandas
- matplotlib
- seaborn
- nltk
- wordcloud
- scikit-learn

You can install all dependencies using:

```bash
pip install -r requirements.txt
```

### Dataset

The dataset contains BBC news articles labeled into categories such as:

- Business
- Entertainment
- Politics
- Sport
- Tech

### Features

- Text preprocessing (tokenization, stopword removal, lemmatization)
- TF-IDF vectorization
- Data visualization (category distribution, word clouds)
- Machine learning model: Random Forest
- Model evaluation (accuracy, precision, recall, F1-score)
- Confusion matrix

### Usage

1. Launch Jupyter Notebook:

```bash
jupyter notebook
```
